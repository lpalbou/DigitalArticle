# Recurrent backlog tasks

This folder contains **recurrent mechanisms** that must be considered whenever their trigger condition applies.

## Why recurrent tasks exist

Some work should not rely on human memory (or an agent “remembering”). Recurrence makes the trigger explicit and auditable.
