# Help & support

## Open Help

- Click the **Help** icon in the top navigation bar.

## Search the documentation

- Go to the **Docs** tab.
- Use the search box to find a topic (e.g., “traces”, “review”, “files”, “export”).

## Read the overview PDF (if available)

- Go to the **PDF** tab.
- Use your browser’s PDF search (Ctrl/⌘+F) to search inside the PDF.

If the PDF is not available on your instance, you can still use the **Docs** tab, or contact support.

## Contact support

- Go to **Help → Contact** and click the email button.
